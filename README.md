# Adaptive Facial Image Deblurring Using CNN-Based Blur Severity Classification and U-Net Restoration

## Overview

Blurring in facial images can significantly degrade the performance of computer vision systems such as face recognition and video surveillance. Traditional image restoration methods often struggle to handle varying levels of blur severity, limiting their effectiveness in real-world scenarios.

This repository presents an adaptive facial image restoration framework that integrates blur severity classification with deep learning-based deblurring. A Convolutional Neural Network (CNN) is first used to classify the blur severity of an input image. Based on the predicted severity level, the image is then processed using a U-Net-based deblurring model specifically optimized for that level of blur.

By dynamically selecting the most appropriate restoration model, the proposed approach improves deblurring performance across a wide range of blur conditions while preserving facial details.

## Method

The proposed pipeline consists of two main components:

1. **Blur Severity Classification**

   * A CNN model classifies blurred facial images into five severity levels:

     * Very Low
     * Low
     * Medium
     * High
     * Very High

2. **Adaptive Deblurring**

   * After classification, a corresponding U-Net-based deblurring model is applied to restore the image.
   * Each model is trained to handle a specific blur level.

This adaptive strategy ensures that each image is restored using the most suitable model for its blur severity.

## Dataset

The experiments use the **Flickr-Faces-HQ (FFHQ)** dataset.

Blurred images are generated by applying Gaussian blur with varying kernel strengths and categorized into five blur severity levels:

* Very Low
* Low
* Medium
* High
* Very High

## Evaluation Metrics

The performance of the restoration models is evaluated using:

* Peak Signal-to-Noise Ratio (PSNR)
* Structural Similarity Index Measure (SSIM)

These metrics measure image reconstruction quality and perceptual similarity to the ground-truth images.

## Results

The proposed adaptive restoration approach consistently outperforms traditional fixed-kernel and multi-kernel deblurring methods across all blur severity levels.

| Blur Level | PSNR (dB) | SSIM  |
| ---------- | --------- | ----- |
| Very Low   | 40.001    | 0.990 |
| Low        | 31.104    | 0.946 |
| Medium     | 27.995    | 0.874 |
| High       | 28.896    | 0.855 |
| Very High  | 26.566    | 0.812 |

These results demonstrate the effectiveness of the proposed method in restoring blurred facial images while maintaining important facial details.
